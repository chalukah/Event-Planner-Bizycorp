‚öôÔ∏è WORKFLOW FOR NEW PANEL EVENT

Phase 1: Initial Outreach (Reshani)

  1. Send E-22 Initial Invitation to potential panelists
  2. Wait 2 days
  3. Send E-20 Follow-up Reminder to non-responders
  4. When panelist confirms, send E-13 Confirmation Thank You
  5. Request panelist details (bio, headshot, contact info)
  6. Hand off to Chaluka

Phase 2: Pre-Event Setup (Chaluka)

  1. Create Zoom webinar and get unique join links for each panelist
  2. Create registration tracking links for each panelist
  3. Create promotional materials (graphics, captions) for each panelist
  4. Draft custom questions for each panelist
  5. Send E-10 Promotional Materials (~10 days before)
  6. Send E-10 Questions (same day or next day after promo materials)

Phase 3: Registration Boost (Chaluka)

  1. Send E-6 Boost Registrations (6 days before)
  2. Send E-5 Help Reach More (5 days before)
  3. Send E-4 Three Days Reminder (3 days before)

Phase 4: Event Day (Chaluka)

  1. Send E-2 Tomorrow Panel (1 day before)
  2. Send E-1 Today is Day (event day morning)
  3. Send E-DAY Starting in 2 Hours (6:00 PM EST)
  4. Send E-DAY Starting Now (7:45 PM EST)
  5. Run panel event at 8:00 PM EST

Phase 5: Post-Event (Chaluka)

  1. Check registration reports for each panelist's tracking link
  2. Get recording link from Zoom
  3. Create attendee list (if panelist brought 10+)
  4. Send E+1 Thank You + Recording (next day)
  5. Remove conditional sections based on registration count:
    • 0-9 registrations: Remove both 10+ and 25+ sections
    • 10-24 registrations: Keep 10+ section, remove 25+ section
    • 25+ registrations: Keep both sections
